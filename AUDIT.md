## Critical — fix first
**Tax-free shopping is wrong, not imprecise.** The file repeatedly tells the user (timeSensitive lines 75-79; checklist chk-do-tax-free lines 2120/2124; planning echoes 1390, 1446; per-shop lines 3124-3432) that tax-free eligibility is tied to a `<6 month` / length-of-stay clock and is lost at `~2026-12-30`. That is the **Temporary-Visitor** rule misapplied to a **work visa**. Since 1 April 2023, tax-free purchasing is restricted to non-residents holding Temporary Visitor / Diplomat / Official status; WHV is work-permitted and explicitly excluded. **A WHV holder is not tax-free eligible at any point, including landing day.** The advice to front-load gear purchases tax-free is money-losing and is flagged 'high confidence' throughout — it must be inverted everywhere it appears. (The Nov-2026 refund-on-departure item at lines 89-93 is itself correct and should be kept.)

## High
- **Moving-in notification penalty understated** (line 35; framing at 331). The `¥50,000` figure is only the Resident Registration Act penalty. For a Residence-Card holder there is a separate immigration-law fine up to `¥200,000` AND status revocation after 90 days unregistered. State both plus the revocation risk.
- **Pension type / tax-agent rationale conflated** (line 98; checklists 2380/2384, 2412/2416). The 20.42% withholding applies only to Employees' Pension (Kosei Nenkin). The file correctly tells the user to enroll in National Pension (Category-1, line 40), which has no such withholding — so the tax-agent step's pension-refund rationale does not apply unless they actually work under an employer's Kosei Nenkin. The residence-tax rationale for the tax agent remains valid.
- **Lump-sum cap missing the 2026 reform** (notes 2416, 2380; event 328/331). Vague 'capped' language predates the cap rising from 60 to 96 contribution months for departures on/after 1 Apr 2026 — which applies to this user (departs ~mid-2027). Add the figure but flag that per-month rates for the 61–96 band were still TBD as of early April 2026 and that one EN source still reports the old number; verify on nenkin.go.jp before filing. Do not hard-code yen amounts.
